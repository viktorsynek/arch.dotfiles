#### Theming
- GTK: Arc-Dark
- Notifications / Lock Screen: Catppuccin Frappé
- SDDM: Catppuccin Frappé Mauve
- Terminal: Ghostty + Oh My ZSH
- Tmux: Catppuccin Frappé
